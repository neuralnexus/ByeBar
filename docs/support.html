GitHub Issues

Bug reports and feature requests are tracked on GitHub. You will need a free GitHub account to open an issue.

  1. Sign in at github.com (or create an account).
  2. Open the link for your request type below. GitHub will open a pre-filled issue form in a new tab.
  3. Fill in the details, then click Submit new issue.

Report a bug · Request a feature · View all issues

What to include

  • ByeBar version (shown in the extension popup)
  • Browser and OS
  • The site URL where the problem happens
  • Steps to reproduce, or a screenshot of the overlay

Privacy

ByeBar does not collect or transmit personal data. Settings stay on your device.
